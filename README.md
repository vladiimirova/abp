# Northline Car Showroom

Адаптивний React SPA для перегляду автомобілів із DummyJSON та AUTO.RIA, фільтрації каталогу, перегляду деталей і додавання локально збережених відгуків.

**Розгорнутий застосунок:** [abp-flax.vercel.app](https://abp-flax.vercel.app/)

## Можливості

- Каталог автомобілів із категорії `vehicle` API DummyJSON
- Актуальні оголошення нових автомобілів AUTO.RIA через захищену Vercel Function
- Пошук, фільтр за рейтингом і сортування за ціною або рейтингом
- Окремі маршрути автомобілів `/vehicles/:vehicleId`
- Характеристики, зображення, наявність, гарантія та відгуки з API
- Валідована форма відгуку з обмеженнями довжини й доступними повідомленнями про помилки
- Збереження користувацьких відгуків окремо для кожного авто в `localStorage`
- Адаптивна семантична верстка для екранів від 420 до 1440 px і більше
- Стани завантаження, помилки, порожнього результату та сторінки 404

## Локальний запуск

```bash
npm install
npm run dev
```

Відкрийте локальну адресу, яку Vite виведе в терміналі.

Під час `npm run dev` Vite запускає локальний серверний endpoint `/api/vehicles`. Для AUTO.RIA створіть локальний файл `.env.local` зі змінною `AUTO_RIA_API_KEY`. Файл ігнорується Git і не потрапляє в репозиторій.

## Налаштування AUTO.RIA на Vercel

1. Відкрийте проєкт у Vercel.
2. Перейдіть до **Settings → Environment Variables**.
3. Додайте `AUTO_RIA_API_KEY` для Production, Preview і Development.
4. Виконайте повторний deployment.

Ключ використовується лише функцією `api/vehicles.js` і не передається у браузер. Не використовуйте для нього префікс `VITE_`. Відповіді AUTO.RIA кешуються на Vercel протягом 15 хвилин.

## Локальний snapshot AUTO.RIA

Щоб один раз завантажити автомобілі та зберегти їх у проєкті:

```bash
npm run sync:ria
```

Команда читає ключ із `.env.local` і записує 6 нормалізованих оголошень у `src/data/riaVehicles.json`. Застосунок використовує snapshot, якщо живий API та браузерний кеш недоступні. AUTO.RIA має погодинні ліміти, тому при помилці `HourOverlimit` повторіть команду після відновлення ліміту.

## Розгортання

Застосунок розгорнуто на Vercel: [https://abp-flax.vercel.app/](https://abp-flax.vercel.app/).

Для самостійного розгортання використовуйте згенеровану директорію `dist`. На статичному хостингу налаштуйте перенаправлення невідомих маршрутів на `index.html`, щоб прямі посилання на кшталт `/vehicles/167` працювали коректно.

## Технічні деталі

Застосунок створено на React, React Router, Vite та чистому CSS без UI-фреймворків.
